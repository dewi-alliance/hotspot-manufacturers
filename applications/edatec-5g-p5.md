## Summary

[EDATEC](https://edatec.cn) is one of [Raspberry Pi's Global Design Partners](https://www.raspberrypi.org/for-industry/design-partners/) who is serving global customers with standard hardware products, custom design and manufacturing services based on Raspberry Pi Technology platform in IoT, Inudstrial Control, Automation & Artificial Intelligenc applications.

We have R&D centers in Shanghai, Shenzhen and Wuhan, we have our production facility, which is based in Dongguan China, and it is also **Raspberry Pi certified manufacturer**.

EDATEC has been working on IoT for years and has several mature IoT Gateway solutions for LoRaWAN and Bluetooth Mesh networks.  We launched LoRa [IoT Gateway](https://edatec.cn/en/Product/Embedded_Computers/2019/0826/77.html) solution in 2018, which is based on Semtech SX1301 and Raspberry Pi Computer Module 3+, we sold this solutions to more than 200 customers in worldwide.

And we designed our own LoRa modules based on SX1302/1303+SX1250.

EDATEC has launched a Full Hotspot [EDA-IoT Indoor Full Hotspot](https://www.eda-iot.com/Products/Hotspot/ED-IoTHotspot-Pi/Overview/) in April 2022. We have also designed multiple Light Hotspots, now we are going to build our new 5G hotspot.

We are glad to introduce **EDA-IoT 5G Hotspot P5** to the Helium community.

## Hardware Information

EDA-IoT 5G Hotspot P5 is a Helium 5G Indoor Hotstpot, but also a **Everything-in-One** Hotspot. It works with Small Cell to provide users with LTE or 5G coverage. We will also keep LoRa based LongFi mining function. It integrates a SX1302/SX1303 LoRaWAN module and ATECC608 security chip, which enable the gateway to do PoC activities and mine IOT and MOBILE tokens at the same time.

* What are your plans for software setup and configuration for the devices?

EDATEC has developed the [EDA-IoT Hotspot App](https://www.eda-iot.com/Guides/Hotspot-App/) which was launched in May 2022. It could operate as the software setup and configuration app. The EDA-IoT Hotspot also has the ability for hosts to change wifi settings.

We provide OTA based software, which can automatically push and install the latest software code into the hotspots.

* Which security implementation (ECC608, TPM, TrustZone, other) are you using?

ECC608

* Which LoRa chipset are you planning to use in your gateway?

SX1302/1303+SX1250

* What is the CPU?

ARM Cortex A72 Quad Core 1.5GHz 64bit CPU

* Additional Hardware Specifications:
  * Concentrator based on the Semtech SX1302/1303
  * Raspberry Pi 4B Inside
    * ARM Cortex A72 Quad Core 1.5GHz 64bit CPU

  * 64GB SD Card
  * Dual Gig-Ethernet and Dual-band Wi-Fi networks

  * Built-in ATECC608 security chip
